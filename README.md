# amazon-price-watcher

Product Advertising API を使って Amazon の商品価格を取得して値下がりをチェックできるようにする

https://affiliate.amazon.co.jp/

面倒なことは [python-amazon-paapi](https://pypi.org/project/python-amazon-paapi/) がやってくれる
